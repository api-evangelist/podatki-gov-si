# OPSI Slovenia (podatki-gov-si)

OPSI Slovenia is a national government open-data portal for Slovenia running **CKAN**, exposing a standardized catalog API over ~8,875 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/podatki-gov-si/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=podatki-gov-si-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.2b

## API
- **OPSI Slovenia CKAN Action API** — base `https://podatki.gov.si/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://podatki.gov.si)
- Live endpoint verified: `https://podatki.gov.si/api/3/action/status_show` · ~8,875 datasets
